# Resume Builder

A modern Flutter application for creating professional resumes. Build your CV with personal details, work experience, education, skills, and add a professional profile photo. 

## Features
* **Profile Photo Integration**: Add your best profile photo via URL to make your resume stand out.
* **Personal Information**: Input your contact details and professional summary.
* **Experience & Education**: Dynamically add your work history and educational background.
* **Skills Management**: List your top professional skills.
* **Live Preview**: See how your resume looks as you build it.
* **Cross-Platform**: Works seamlessly across Web, Mobile, and Desktop.

## Tech Stack
* Flutter
* Dart

## Setup
1. Ensure you have Flutter installed.
2. Run `flutter pub get` to fetch dependencies.
3. Run the app using `flutter run`.
